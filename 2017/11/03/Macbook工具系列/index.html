Macbook工具系列

工欲善其事，必先利其器。作为一个工具控，从买 Macbook 开始折腾到现在，几个月了，终于决定记录一下自己在这个平台收集的工具集。一来是分享，二来是做个总结梳理，如果以后系统重置的话，也方便重新搭建自己的环境。以下记录的这些工具，至少现在来说，绝对是最牛逼的！如果你觉得某些不是，那么没关系，但他是最适合我的…因为每种同类型的工具我都是经过细细品味后精挑细选出来的。废话不多说，上干货！

实用小工具

  • ShadowsocksX-NG-R8

    我的命根，不解释，时时刻刻离不开它，每台 PC 装的第一个软件

  • Alfred 3

    一个绝对可以代替你的 Spotlight 的工具，而且强了不知多少倍。

  • Moom

    一个可以对窗口进行快捷移动，缩放，分屏的工具

  • TouchSwitcher

    让你的 TouchBar 变成 TouchBar Plus

  • CheatSheet

    快捷键太多记不住？没关系，用这个，长按 cmd 就可以显示当前应用的所有快捷键

  • AppCleaner

    帮助在卸载软件时自动删除配置和缓存文件

  • iStat Menus

    一个显示系统信息的状态栏工具，我主要用的是网速和温度，天气虽然也是极好的，但是有些问题

  • SpaceLauncher

    通过空格键+其他键的方式快速启动/切换应用

  • Vanilla

    状态栏图标太多？它可以帮你隐藏！由于我是早期试用用户，所以有作者给的永久授权，赚~

  • DosBox

    这个东西其实我只用它来干一件事，没事敲一敲 tt5（一款练习打字的祖师爷级别的终极武器）

  • HyperSwitch

    当你按 cmd + Tab 来切换应用程序的时候，会不会觉得系统自带的功能还是有点鸡肋？没错，如果一个应用的窗口隐藏起来了，你还需要按住 Opt 来唤起它，用这个工具就很爽了…

  • Irvue

    一个自动更换壁纸的软件，图片都来自 Unsplash ，最完美！

  • Itsycal

    一个状态栏日历软件，简约小巧，我喜欢！

  • 电池监控器

    在状态栏就可以看到电池的循环次数，健康度等信息

开发工具

  • API 工具： Dash

  • 编辑器： Sublime Text、Visual Studio Code、IntelliJ IDEA、WebStorm

  • Scrum： Trello

  • 版本控制： Gitkraken（其实用命令行比较多，这个只是备用）

  • 数据库： Sequel Pro

  • 文件对比： Beyond Compare

  • 虚拟化： Docker

  • 命令行： iTerm2、oh my zsh

其他日常使用

  • 视频： IINA

  • 音乐： 网易云音乐、QQ音乐（因版权问题，虽然网易云比较舒心，但是很多歌没有，这两个的话，基本上都有了）

  • 邮件： Spark

  • 网盘： Dropbox、Resilio Sync

  • 输入法： Rime

  • 思维导图： MindNode

  • 阅读： Pocket、Reeder、一览

  • 词典： 有道

  • UI： Sketch

  • 解压： BetterZip

  • 下载： Folx

  • 图片压缩： Compress

  • MarkDown 编辑器： Typora

  • iPhone： iTools

  • 笔记： Notion
